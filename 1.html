Web

  A website[1] is a collection of related web pages, including multimedia content,
 typically identified with a common domain name, and published on at least one web server.
 A website may be accessible via a public Internet Protocol (IP) network, such as the Internet,
 or a private local area network (LAN), by referencing a uniform resource locator (URL) that identifies the site.
 Hypertext Markup Language (HTML) is the standard markup ther items.
 HTML elements are delineated by tags, written using angle brackets.
